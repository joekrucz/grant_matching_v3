# Funding Source Logos

This directory contains logo files for all funding sources.

## Required Logo Files

Place the following logo files in this directory:

- `ukri.svg` (or `.png`) - UKRI logo
- `bbsrc.svg` - BBSRC logo
- `epsrc.svg` - EPSRC logo
- `mrc.svg` - MRC logo
- `stfc.svg` - STFC logo
- `ahrc.svg` - AHRC logo
- `esrc.svg` - ESRC logo
- `nerc.svg` - NERC logo
- `nihr.svg` - NIHR logo
- `catapult.svg` - Catapult Network logo
- `innovate_uk.svg` - Innovate UK logo

## Logo Sources

- **UKRI Councils**: Download from [UKRI Brand Guidelines](https://www.ukri.org/wp-content/uploads/2020/10/UKRI-050920-BrandGuidelines.pdf)
- **NIHR**: Check [NIHR website](https://www.nihr.ac.uk/) media/brand resources
- **Catapult Network**: Check [Catapult website](https://catapult.org.uk/) brand guidelines
- **Innovate UK**: Included in UKRI brand guidelines

## File Format

- **Preferred**: SVG format (scalable, smaller file size)
- **Fallback**: PNG format (24x24px or 32x32px minimum, transparent background)
- **Naming**: Use lowercase source code (e.g., `bbsrc.svg`)

## Brand Guidelines

Ensure all logo usage complies with each organization's brand guidelines. Contact the organizations directly if you need official logo files.

## Note

The application gracefully handles missing logo files - if a logo doesn't exist, the image will be hidden and only the source name will display.
